## Color Coding

- **Blue Badges**: Version numbers (v1, v2, v3)
- **Green Badge**: "Current" version indicator
- **Gray Text**: Timestamps and word counts
- **Italic Gray**: Feedback text
- **Primary Blue Button**: Main action (Regenerate Chapter)
- **Icon Buttons**: Restore (â†©ï¸) and View (ğŸ‘ï¸)

## Responsive Behavior

1. **Version History auto-updates** when:
   - User selects a different chapter
   - User successfully regenerates a chapter
   - User restores a previous version

2. **Feedback textarea clears** after successful regeneration

3. **Version viewer dialog** is full-screen maximized for easy reading

4. **All interactions show notifications** for user feedback

5. **Generation can be cancelled** just like normal chapter writing
